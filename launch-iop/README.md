# Launch IOP — Website

Built with Astro. Deploys automatically on Vercel.

## Pages
- / (home), /program, /how-it-works, /team, /faq, /contact

## SEO/AIEO built in
- Per-page titles, meta descriptions, canonicals, Open Graph
- Sitewide MedicalClinic structured data; FAQPage schema on /faq; Person schema on /team
- Auto-generated sitemap.xml + robots.txt

## To run locally (optional)
npm install
npm run dev   # http://localhost:4321

## To deploy
Push this folder to GitHub, then import the repo at vercel.com. Vercel auto-detects Astro.
